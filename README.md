# Spellchecker

Project to implement all Data Structures in a single project

C++ program that takes as a command line argument the name of an input text file 
containing some text in English and checks to see if it has any spelling errors, 
in which case it gives suggestions of alternative words to use. 

More info on: -> https://arielortiz.info/apps/s202213/tc1031/integrating_project/
